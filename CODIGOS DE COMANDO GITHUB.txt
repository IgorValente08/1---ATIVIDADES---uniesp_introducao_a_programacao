git init (iniciar um respositório)

git status (verificar atualização)

git add "nome do arquivo.txt" (adicionar arquivo no repositório)

git add . (adicionar todos os aqruivos no respositório)

git commit -m "commit inicial" (criar versões dos códigos)

git remote add origin (ADICIONAR PARA QUAL REPEPOSITÓRIO SERÁ ENVIADO OBS: COLOCAR O LINK DO RESPOSITÓRIO AO LADO)

git push --set-upstream origin master (Subir versão para o GIT HUB)

git reflog (verificar versões que foram inseridas)

git reset --hard
